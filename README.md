臺鐵列車時刻查詢 LINE 機器人: 以查詢指定日期與車站之列車資料
===

請用 QR Code 掃描並且加入為好友

![](images/QR.png)

## 主要功能:

1. 目前支援顯示指定列車資訊，包括車次, 時間, 山海線, 與可否訂票
2. 格式檢查
3. 新式樣回應訊息

    <img src="images/trainbot.gif" alt="Cover" width="30%"/>
